# Snake-Game
Jogo desenvolvido baseado no clássico Snake, para fins didáticos.

**Comandos:**
- Seta para cima;
- Seta para baixo;
- Seta para esquerda;
- Seta para direita.

**Regras:**
- Não bater na parede;
- Não bater em si mesmo.

**Objetivo:** 
- Coletar as comidas que forem geradas.

**Desafio:**
- Fazer 1.000.000 de pontos.
